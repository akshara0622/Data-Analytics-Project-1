Data Analytics Project 1 - Data Cleaning & Preparation

##Objective##

Clean a raw dataset by handling missing values, verifying duplicate records, and preparing the data for analysis.

##Tasks Performed##

1). Loaded the dataset using Python and Pandas.

2). Checked for missing values across all columns.

3). Identified missing values in the CouponCode column.

4). Filled missing CouponCode values with "No Coupon".

5). Checked for duplicate rows and duplicate records.

6). Verified the final dataset contained no missing values and no duplicates.

7). Exported the cleaned dataset to a new Excel file.

ii. Pandas
iii. VS Code
